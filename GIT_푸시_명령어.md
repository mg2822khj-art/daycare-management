# Git 푸시 명령어

## 📤 GitHub에 변경사항 업로드하기

프로젝트 폴더에서 **PowerShell**을 열고 다음 명령어를 순서대로 실행하세요:

### 1단계: 변경사항 확인
```powershell
git status
```

### 2단계: 모든 변경사항 추가
```powershell
git add .
```

### 3단계: 커밋 메시지 작성
```powershell
git commit -m "500 오류 수정 및 기존 데이터 호환성 개선"
```

### 4단계: GitHub에 푸시
```powershell
git push
```

---

## 🚀 한 번에 실행하기

위의 명령어를 한 번에 실행하려면:

```powershell
git add . && git commit -m "500 오류 수정 및 기존 데이터 호환성 개선" && git push
```

## ⚠️ 주의사항

1. **첫 푸시인 경우**: 원격 저장소가 설정되어 있는지 확인하세요.
2. **인증 오류**: GitHub 인증이 필요할 수 있습니다.
3. **충돌 발생**: 다른 곳에서 수정한 내용이 있으면 먼저 `git pull`을 실행하세요.
